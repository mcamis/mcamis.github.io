Drive the Vote

Getting out the vote

Javascript Development

Drive the Vote connects voters and volunteer drivers during elections. Organized via DevProgress, the development team consisted entirely of remote volunteers. My role was the development of a mobile web app for drivers using React, Redux and Webpack. After kicking off the project in July we successfully launched in time for the 2016 election.

  • React.js
  • Redux.js
  • Javascript/ES6
  • HTML/CSS

Using the browser's geolocation API, the app pairs drivers with voters in their general area. Drivers then use an intuitive interface for completing rides with additonal options to use native applications for naviagtion and communication with riders.
